##  **Huella de carbono digital**

Es el impacto que tiene el uso de software, servidores, centro de datos y más, esto se debe que al crear cada vez software más modernos, hay mayores gastos de energía y emisiones, para evitar esto lo mejor que se puede hacer es optimizar el código y reducir en gastos  
   
Esto se usa mucho en estrategias de Green it, sobre todo dentro de las empresas tecnológicas, Como son Google

##  **Accesibilidad universal**

Consiste en diseñar aplicaciones y webs  que puedan usar cualquier dispositivo de cualquier persona, Para conseguir esto, primero se debe reducir la cantidad de recursos que utilizan las aplicaciones modernas, pues gracias a esto, los dispositivos antiguos se quedan anticuados
